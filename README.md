# polishc
C. Teraz dobre, bo polske. Polskie C.

Zestaw makr, które są swoistym prztyczkiem w nos programistycznym purystom, którzy nie cierpią Ponglishu w kodzie ;)

---

> [!NOTE]
> Polskie C w głównej mierze jest ciekawostką - projektem satyrycznym, albo "poważnym shitpostem", który jest rozwijany kiedy tylko nadarzy się ku temu okazja. Działa, jest niekompletny, ale jakoś działa. PR'y są mile widziane tak samo jak issue'y, lecz bez gwarancji szybkiej reakcji.
> Z racji na powagę projektu (bądź jej brak), wszelkie wsparcie również jest (mocno) ograniczone.
> 
> **Używasz Polskiego C na własną odpowiedzialność!**

> [!NOTE]
> Polskie C nie jest dialektem języka C i nie powinno być traktowane jako takowy.
